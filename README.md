# 马里奥跑酷大冒险 🎮

一款使用Python和Pygame开发的2D横版跑酷游戏,灵感来自经典的超级马里奥。

## 游戏特色

- 🏃 马里奥风格的主角设计
- 🐢 三种可爱的敌人:乌龟、兔子、蘑菇
- 🌈 自动滚动的背景效果
- ⭐ 简单有趣的计分系统
- 🎯 通关目标:躲避10个敌人

## 游戏玩法

### 操作方式
- **空格键** 或 **↑方向键**: 跳跃
- **R键**: 重新开始(游戏结束后)
- **Q键**: 退出游戏

### 游戏规则
1. 主角自动向前跑动,背景自动滚动
2. 敌人从右侧不断出现
3. 使用跳跃躲避敌人
4. 每成功躲避1个敌人得1分
5. 碰到敌人即游戏失败
6. 累计躲避10个敌人即通关胜利

## 安装说明

### 环境要求
- Python 3.7 或更高版本
- Pygame库

### 安装步骤

1. **克隆或下载项目文件**

2. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

   或者直接安装Pygame:
   ```bash
   pip install pygame
   ```

3. **运行游戏**
   ```bash
   python mario_runner.py
   ```

## 技术特点

- **面向对象设计**: 使用类封装玩家、敌人、装饰物等游戏对象
- **精灵系统**: 利用Pygame的sprite模块进行碰撞检测和渲染
- **动态难度**: 敌人随机生成,增加游戏挑战性
- **视觉反馈**: 清晰的UI显示分数和操作提示
- **流畅动画**: 60FPS帧率保证游戏流畅运行

## 自定义修改

你可以轻松修改以下参数来调整游戏难度:

在 `mario_runner.py` 文件中:
- `SCROLL_SPEED`: 修改滚动速度(默认5)
- `JUMP_STRENGTH`: 修改跳跃高度(默认-15)
- `GRAVITY`: 修改重力大小(默认0.8)
- 第268行: 修改敌人生成频率
- 第10行: 修改通关分数(默认10)

## 常见问题

**Q: 游戏窗口无法打开?**  
A: 确保已正确安装Pygame库,可尝试重新安装: `pip install --upgrade pygame`

**Q: 游戏运行卡顿?**  
A: 关闭其他占用资源的程序,或降低FPS值

**Q: 如何调整难度?**  
A: 修改上述参数,增加`SCROLL_SPEED`或减少敌人生成间隔可提高难度
